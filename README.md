# Quiz App (Node.js)

This is a simple command-line Quiz Application built using JavaScript and Node.js.  
The program asks multiple-choice questions and keeps track of the user's score.

## Features

- 20 multiple choice questions
- Score tracking
- Runs in terminal
- Written in JavaScript (Node.js)
- Uses modular files (quiz.js, questions.js)
- Good practice for backend / fullstack development

## Technologies Used

- JavaScript
- Node.js
- Git
- GitHub

## How to Run

1. Install Node.js
2. Clone the repository

git clone https://github.com/yourusername/quiz-app.git

3. Go to folder

cd quiz-app

4. Run the program

node quiz.js
